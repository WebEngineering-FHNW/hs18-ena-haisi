Poster Art

About

In appreciation of Swiss Style and Swiss Typography this project is be the place to learn about famous artists and their works - specifically posters.

User Area

Displays all uploaded posters and their information in a grid system.

Using CSS grid-layout and flexbox-layout was a major focus of this project in order to increase my knowledge on these subjects.

Admin Page

The admin page is used to do all CRUD operations.

This side of the web-app is solly used for functionality reasons and is therefor purely functional.
